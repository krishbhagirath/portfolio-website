# Krish Bhagirath – Portfolio

A fast, responsive personal site for projects, experience, and contact info.

## Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui + Radix

## Getting started
```bash
npm install
npm run dev
```
Then open the local URL from the terminal.

## Layout
- `src/components` – UI sections and shared components (shadcn/ui under `ui/`)
- `src/content` – Copy and metadata for hero, skills, projects, experience, etc.
- `src/pages` – Routes (`Index.tsx`, `NotFound.tsx`)

## Notes
- Deployed on Vercel: https://krish-bhagirath.vercel.app
- Static assets live in `public/`.
